# Dominik Portfolio

React + Vite + Tailwind portfolio website.

## Local setup

```bash
npm install
npm run dev
```

## Images

Place your images inside:

```text
public/images/
```

The code expects image names like:

```text
IMG_8890 2.jpg
image000000.JPG
IMG_0407.jpg
```

## Deploy

Upload the full project folder to GitHub, then import the repository into Vercel.
